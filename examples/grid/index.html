Njörðr's grid system is only mixin's

You can create a Boostrap-like class-based grid system yourself, as in this example CSS. But I prefer not to do that, as I rarely use grids at all - and I don't really like them. So try to just use the mixins!

There's no media queries built into this grid system on purpose, as that leaves it up to the single project to determine how to best control the layout with media queries.
